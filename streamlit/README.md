# Demo_Streamlit_Apps
This repository is using for demo Streamlit Web Apps.

# To use this demo:
***Demo Files:***
* ***Command to run streamlit app.***
>rasa train
>rasa --enable-api
>cd streamlit
>streamlit run file_name.py
